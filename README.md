# Spotide/行きたい場所リスト共有アプリ

## サービスの概要  
「Spotide（スポティード）」は、旅行やおでかけの計画をスムーズにするための店舗検索・共有アプリです。  
Googleマップと連携し、店舗を簡単に検索、リスト化、グループで共有、投票まで一貫して可能です。


## サービスのURL
https://spotide.vercel.app/login  
リストの作成はGoogleアカウントでログインすることで可能です。  
リストの参加は未ログインでも可能です。
